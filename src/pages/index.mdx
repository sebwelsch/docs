## Idura Verify

Idura Verify provides eID authentication as a service.

If you're new to Idura Verify and electronic identities (eIDs), start with our [Getting Started](/verify/getting-started/overview) guide.

All eID services support user authentication. With Idura Verify, you integrate using _OpenID Connect_ (aka OIDC). For a general overview of how OIDC works with Idura Verify, see [Using OIDC](/verify/getting-started/oidc-intro).

[Idura Verify Documentation](/verify)

## Idura Signatures

Idura Signatures allows you to sign any PDF document using all the [national eIDs](/verify/e-ids/) supported by Idura Verify.

Documents will be produced according to the [PAdES-LTA standard](https://www.criipto.com/blog/pades-signature-levels-explained).
